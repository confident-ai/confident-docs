## Deployed services

The Kubernetes cluster runs the following Confident AI services:

| Service                | Description                                            |
| ---------------------- | ------------------------------------------------------ |
| **confident-backend**  | Express.js API service handling core platform logic    |
| **confident-frontend** | Next.js web application for the Confident AI dashboard |
| **confident-evals**    | FastAPI service for running LLM evaluations            |
| **confident-otel**     | OpenTelemetry collector for trace ingestion            |
| **redis**              | In-memory cache for session and queue management       |
