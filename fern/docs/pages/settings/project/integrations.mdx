## Set Up an Integration

To configure an integration:

1. Navigate to **Project Settings** â†’ **Integrations**
2. Select an integration from the dropdown
3. Provide the required webhook URL (or email address for email notifications)
4. Click **Save** to apply your changes

## Available Integrations

- Slack (learn how to create a webhook [here](https://api.slack.com/messaging/webhooks))
- Discord (learn how to create a webhook [here](https://support.discord.com/hc/en-us/articles/228383668-Intro-to-Webhooks))
- Teams (learn how to create a webhook [here](https://learn.microsoft.com/en-us/microsoftteams/platform/webhooks-and-connectors/how-to/add-incoming-webhook?tabs=newteams%2Cdotnet))
- Email

For any integrations that are not covered, email support@confident-ai.com.
