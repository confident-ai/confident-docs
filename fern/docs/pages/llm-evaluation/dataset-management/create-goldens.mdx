## Overview

A dataset, single or multi-turn one, is a list of goldens and forms the basis of any evaluation workflow in development. In this section, you'll learn to manipulate goldens in datasets, including:

- Uploading goldens via CSV on the platform
- Build an automated golden ingestion pipline via the Evals API
- Assiging different team members to review and finalize goldens

<Tip>
  If you haven't already, you should get yourself familiarized with [what are
  goldens.](/llm-evaluation/core-concepts/test-cases-goldens-datasets#goldens)
</Tip>

## Manage Goldens via Evals API

If you wish to upload goldens programmatically instead, you can leverage Confident AI's Evals API. You can either **push** goldens in the "finalized" state, or **queue** them to mark them "unfinalized".

<Note>Only finalized goldens will be pulled for evaluation.</Note>

### Push goldens

If the dataset does not already exist, Confident AI will create it for you.

<Tabs>

<Tab title="Python" language="python">

For **single-turn** datasets, push single-turn goldens:

```python main.py
from deepeval.dataset import EvaluationDataset, Golden

goldens = [Golden(input="How tall is Mr. Everest?")]
dataset = EvaluationDataset(goldens=goldens)

dataset.push(alias="YOUR-DATASET-ALIAS")
```

For **multi-turn** datasets, push multi-turn goldens:

<Tabs>
<Tab title="With Turns" language="turns">
```python main.py
from deepeval.dataset import EvaluationDataset, ConversationalGolden
from deepeval.test_case import Turn

goldens = [
    ConversationalGolden(
        scenario="Angry user asking for a refund.",
        turns=[Turn(role="user", content="Give me my money!")]
    )
]
dataset = EvaluationDataset(goldens=goldens)

dataset.push(alias="YOUR-DATASET-ALIAS")
```
</Tab>
<Tab title="Without Turns" language="no-turns">
```python main.py
from deepeval.dataset import EvaluationDataset, ConversationalGolden

goldens = [ConversationalGolden(scenario="Angry user asking for a refund.")]
dataset = EvaluationDataset(goldens=goldens)

dataset.push(alias="YOUR-DATASET-ALIAS")
```
</Tab>
</Tabs>

</Tab>

<Tab title="Typescript" language="typescript">

### Queue goldens

If the dataset does not already exist, Confident AI will create it for you.

<Tabs>

<Tab title="Python" language="python">

For **single-turn** datasets, queue single-turn goldens:

```python main.py
from deepeval.dataset import EvaluationDataset, Golden

goldens = [Golden(input="How tall is Mr. Everest?")]
dataset = EvaluationDataset()

dataset.queue(alias="YOUR-DATASET-ALIAS", goldens=goldens)
```

For **multi-turn** datasets, queue multi-turn goldens:

<Tabs>
<Tab title="With Turns" language="turns">
```python main.py
from deepeval.dataset import EvaluationDataset, ConversationalGolden

goldens = [
    ConversationalGolden(
        scenario="Angry user asking for a refund.",
        turns=[Turn(role="user", content="Give me my money!")]
    )
]
dataset = EvaluationDataset()

dataset.queue(alias="YOUR-DATASET-ALIAS", goldens=goldens)
```
</Tab>
<Tab title="Without Turns" language="no-turns">
```python main.py
from deepeval.dataset import EvaluationDataset, ConversationalGolden

goldens = [ConversationalGolden(scenario="User asking for a refund.")]
dataset = EvaluationDataset()

dataset.queue(alias="YOUR-DATASET-ALIAS", goldens=goldens)
```
</Tab>
</Tabs>

</Tab>

<Tab title="Typescript" language="typescript">

## Custom Dataset Columns

You can add custom columns to a dataset to hold additional additation data for each golden as long as they don't clash with any of the existing default field names (e.g. "Input", "Actual Output", etc.)

<Frame caption="Create Custom Columns" background="subtle">

<video
  autoPlay
  loop
  muted
  src="https://confident-docs.s3.us-east-1.amazonaws.com/datasets:new-column-4k.mp4"
  type="video/mp4"
/>

</Frame>

You can also do it through the Evals API when pushing or queueing goldens by including the custom column key values field in single/multi-turn goldens:

<Tabs>

<Tab title="Python" language="python">

```python main.py
from deepeval.dataset import Golden, ConversationalGolden

golden = Golden(custom_column_key_values={"Key": "Value"}, ...)
multiturn_golden = ConversationalGolden(custom_column_key_values={"Key": "Value"}, ...)
```

</Tab>

<Tab title="Typescript" language="typescript">

```ts index.ts
import { Golden, ConversationalGolden } from "deepeval-ts";

const golden = new Golden({ customColumnKeyValues: { Key: "Value" }, ... });
const multiturnGolden = new ConversationalGolden({
  customColumnKeyValues: { Key: "Value" }, ...
});
```

</Tab>
