## Overview

A dataset, single or multi-turn one, is a list of goldens and forms the basis of any evaluation workflow in development. In this section, you'll learn to manipulate goldens in datasets, including:

- Uploading goldens via CSV on the platform
- Build an automated golden ingestion pipline via the Evals API
- Assiging different team members to review and finalize goldens

<Tip>
  If you haven't already, you should get yourself familiarized with [what are
  goldens.](/llm-evaluation/core-concepts/test-cases-goldens-datasets#goldens)
</Tip>

## Delete Dataset

You can delete a dataset directly on the platform.

<Warning>
  This action cannot be undone. All goldens or conversational goldens in the
  dataset will be permanently deleted.
</Warning>

<Frame caption="Delete Dataset on Confident AI" background="subtle">

<video autoPlay loop muted data-video="datasets.delete" type="video/mp4" />

</Frame>

You can also do it programmatically through the Evals API.

<Tabs>

<Tab title="Python" language="python">

```python main.py
from deepeval.dataset import EvaluationDataset

dataset = EvaluationDataset()
dataset.delete(alias="YOUR-DATASET-ALIAS")
```

</Tab>

<Tab title="Typescript" language="typescript">

```ts index.ts
import { EvaluationDataset } from "deepeval-ts";

const dataset = new EvaluationDataset();
dataset.delete({ alias: "YOUR-DATASET-ALIAS" });
```

</Tab>
