## Overview

Projects are isolated workspaces for your datasets, prompts, traces, and evaluations. With the `ConfidentClient` you can provision and tear down projects in code. This is especially useful for spinning up a project per AI agent, environment, or customer.

<Note>

All project methods require an **Organization API Key**. See [Setup](/docs/management/introduction#setup) to create a client.

</Note>

## List Projects

You can list every project in your organization.

<Tabs>

<Tab title="Python" language="python">

```python
from deepeval.confident import ConfidentClient

client = ConfidentClient()

projects = client.list_projects()
for project in projects:
    print(project.id, project.name)
```

</Tab>

<Tab title="TypeScript" language="typescript">

```typescript
import { ConfidentClient } from "deepeval";

const client = new ConfidentClient();

const projects = await client.listProjects();
projects.forEach((project) => console.log(project.id, project.name));
```

</Tab>

</Tabs>

## Create a Project

You can create a new project with just a `name`, while the `description` is optional.

<Tabs>

<Tab title="Python" language="python">

```python
project = client.create_project(
    name="Customer Support Bot",
    description="Production support assistant",
)
print(project.id)
```

</Tab>

<Tab title="TypeScript" language="typescript">

```typescript
const project = await client.createProject({
  name: "Customer Support Bot",
  description: "Production support assistant",
});
console.log(project.id);
```

</Tab>

</Tabs>

## Get a Project

You can retrieve a single project by its `project_id`.

<Tabs>

<Tab title="Python" language="python">

```python
project = client.get_project(project_id="proj_123")
```

</Tab>

<Tab title="TypeScript" language="typescript">

```typescript
const project = await client.getProject("proj_123");
```

</Tab>

</Tabs>

## Update a Project

You can update a project's `name`, `description`, or both, and only the fields you pass are changed.

<Tabs>

<Tab title="Python" language="python">

```python
project = client.update_project(
    project_id="proj_123",
    name="Support Bot (v2)",
)
```

</Tab>

<Tab title="TypeScript" language="typescript">

```typescript
const project = await client.updateProject("proj_123", {
  name: "Support Bot (v2)",
});
```

</Tab>

</Tabs>

## Delete a Project

You can permanently delete a project from your organization.

<Warning>

Deleting a project permanently removes all of its datasets, prompts, traces, and evaluations. This cannot be undone.

</Warning>

<Tabs>

<Tab title="Python" language="python">

```python
client.delete_project(project_id="proj_123")
```

</Tab>

<Tab title="TypeScript" language="typescript">

```typescript
await client.deleteProject("proj_123");
```

</Tab>

</Tabs>
