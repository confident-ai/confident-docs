**Prompts** are either text or a list of text templates that guide LLM behavior by providing instructions, context, and structure for generating responses. Refer to the [concepts page](/concepts/hyperparameters#prompts) for more information.

**Datasets** in Confident AI is a collection of [test cases](/concepts/test-cases) and/or [goldens](/concepts/datasets#goldens) that you use to evaluate your LLM application. Refer to the [datasets concept page](/concepts/datasets) for more information.


# Getting Started with Prompts and Datasets

A prompt and dataset are how you run evaluations on Confident AI.
Prompts will be used to provide the input to your LLM application, and datasets will be used to store the inputs and expected outputs for evaluation.

By [prompt versioning](/concepts/hyperparameters#prompt-versioning), we track which prompt template performs best, while annotating datasets enables effective evaluations.

### Create a Prompt

Navigate to your project, and go to the **Prompt Studio** tab on the left navigation drawer. Click on **Create a Prompt**, and provide your prompt with an alias. For example, the alias of your first prompt can be called `System Prompt`, which is where you can centrally store your [system prompt](https://promptengineering.org/system-prompts-in-large-language-models/), instead of using CSV files, or `.txt` files without any [versioning](/concepts/hyperparameters#prompt-versioning).

### Create a Prompt Version

Now that you have a prompt, create a first version of your prompt. You should paste in the prompt that you are currently using in your LLM application. For example, you can paste in your entire system prompt and click **Save**.

### Using Variables in Prompts

When your prompt needs to include dynamic content, you can use variables to create a [prompt template](/concepts/hyperparameters#prompt-templates). Variables can be added by wrapping them in spaced out double curly braces. Variables must have no spaces in variable name. Here's how to format them correctly:

```text showLineNumbers
# ✅ Correct usage:
"Hi, my name is {{ name }}."
"The temperature is {{ temperature }} degrees."
"User input: {{ user_input }}"

# ❌ Incorrect usage:
"Hi, my name is {{ variable name }}."  # Spaces in variable name
"Hi, my name is {{  variable_name }}." # Extra spaces around variable name
```

You can also create **a list of prompt messages**, or which is a list of prompts with the `"role"` and `"content"` field present. If you decide to create a prompt message instead, you can supply it directly to your LLM provider once pulled from Confident AI (more info on this later). More on prompt messages can be [found here.](/prompt-studio/prompt-versioning#commit-a-prompt-message-list)

### Create a Dataset

Navigate to your project, and go to the **Datasets** tab on the left navigation drawer. Click on **Create a dataset**, and provide your dataset with an alias.

### Create or Upload Golden(s)

Once you've created a dataset, you'll be directed to the **Dataset Editor** page for your newly created dataset. Depending on whether you already have [goldens](/concepts/datasets#goldens), you should either:

1. Upload a CSV file of your goldens to your dataset, if you already have a dataset prepared.
2. Create a golden, with the input as the text you typically prompt your LLM application with for testing.

If you don't already have a dataset, create a golden with a text input text you usually use to test your LLM application. You can add more goldens later to the dataset.
